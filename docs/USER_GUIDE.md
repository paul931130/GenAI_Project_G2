# 詩境 使用者指南

## 1. 開始使用

開啟網站後會看到「通靈問詩」頁面。你可以直接輸入作者、詩名、詩句片段或意境描述。

可測試輸入：

- `靜夜思`
- `石壕吏`
- `床前明月光`
- `邊塞悲壯`

## 2. 本機詩庫模式

如果沒有輸入 Gemini API Key，系統會使用本機 `poems.json` 詩庫。本機詩庫目前主要收錄唐詩，因此輸入唐詩篇名、作者或名句時較容易命中。

如果輸入宋詞或較抽象的意境，例如「蘇軾的中秋詞」，本機詩庫可能查不到，這時可以使用 Gemini API。

## 3. 隨機召喚

在「焚香問詩」按鈕下方有「🎲 隨機一首」。按下後系統會直接從本機 `poems.json` 抽出一首詩。

這個功能：

- 不需要 Gemini API Key
- 不消耗 API 額度
- 適合展示與快速探索詩庫

## 4. 免費瀏覽器朗讀

查到詩後，詩作原文旁會出現「朗讀」按鈕，詩魂自語區也有獨立「朗讀」按鈕。

朗讀功能使用瀏覽器內建語音，不需要 ElevenLabs、不需要語音 API key，也不需要後端伺服器。

如果按下後沒有聲音，請檢查：

- 瀏覽器是否支援語音朗讀
- 系統音量是否開啟
- 手機是否靜音
- 是否使用 Chrome 或 Edge 測試

## 5. 加入 Gemini API Key

點左側的 `API KEY` 按鈕，貼上 Gemini API Key。系統會在輸入停止後延遲自動測試一次，用來確認 key 是否可用。

安全設定：

- Key 只存在目前分頁的 `sessionStorage`。
- 關閉分頁後 Key 會清除。
- Key 不會寫進 `index.html`、`app.js` 或 GitHub。
- 不輸入 key 仍可使用本機詩庫、隨機召喚與免費朗讀。

## 6. API 狀態

左側會顯示目前狀態：

- 已連線：Gemini API key 已通過測試，可用於 AI 生成。
- 僅本機：未輸入 key、key 不可用，或目前使用本機資料。
- 配額受限：API key 測試或請求遇到 `Too many requests / 429`，系統會暫停自動測試 60 秒，避免重複消耗請求。

## 7. 查詢結果

查到詩後，畫面會顯示：

- 詩名、作者、朝代
- 詩作原文
- 來源：Gemini 或本機詩庫
- 白話釋義
- 情緒分析
- 創作背景與典籍來源

你可以使用「字小／字大」調整詩文與篇名字級，也可以按「複製」取得格式化的詩名、作者朝代與原文。

## 8. 收藏詩作

按「收藏」後，詩作會存到瀏覽器 `localStorage`。左側「詩藏」會顯示收藏數量，點進去可以重新開啟已收藏的作品。

本版已移除重複的獨立「詩藏」按鈕，保留主分頁中的詩藏入口。

## 9. 向詩魂提問

在查詢結果下方可以輸入問題，例如：

- 這首詩在說什麼？
- 主要情緒是什麼？
- 有哪些修辭？
- 背景是什麼？

如果 Gemini API 可用，系統會以詩人第一人稱回答；若 API 不可用，會以本機資料產生簡短回覆。

## 10. 雙魂對話

到「雙魂對話」頁面，選擇兩位詩人與一個主題，例如「月亮」、「離別」、「人生無常」。系統會讓兩位詩人輪流對話。

如果選到同一位詩人，系統會提醒你改選不同人物。

## 11. 常見問題

### 為什麼水調歌頭本機查不到？

目前本機 `poems.json` 主要是唐詩，宋詞不一定收錄。可以使用 Gemini API 查詢。

### 為什麼出現配額受限？

Gemini API 可能短時間請求太多，或 key 的免費額度已用完。可以稍後再試，或先使用本機詩庫。

### 可以把 API Key 寫進程式碼嗎？

不建議。公開到 GitHub 後 key 會被任何人看見，可能造成配額被用光。此專案採用 sessionStorage 暫存。

### 為什麼不能直接雙擊 index.html？

因為本機詩庫是用 `fetch("poems.json")` 載入，直接用 `file://` 開啟時，瀏覽器可能會阻擋讀取。請使用 `python -m http.server 8000` 或 VS Code Live Server 測試。
