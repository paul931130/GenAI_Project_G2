# Release Notes

## v1.4.0｜校注分離與快取修正

### 更新重點

1. 修正原詩正文混入校勘註記的問題。
   - 例如舊版可能顯示：`孤帆遠影碧空盡，唯見長江天際流。 (唯 通：惟)`
   - 新版會顯示為：`孤帆遠影碧空盡，唯見長江天際流。`
   - 校注區另顯示：`唯，通「惟」。`

2. 新增「校注／異文」自動分離機制。
   - 支援 `(通：)`、`(一作：)`、`(又作：)`、`(版本...)` 等常見格式。
   - 原詩直排區只保留真正詩句。

3. 修正手機版直排穩定性。
   - 校注改為橫排、小字、低權重顯示。
   - 避免括號註記或長註解撐開詩文白框。

4. 修正本機詩庫快取問題。
   - `poems.json` 讀取時自動加入版本與時間戳。
   - 不需要使用者手動在網址後面加 `?v=`。

5. 更新前端資源版本。
   - `index.html` 已更新 `app.js` 與 `style.css` 的版本參數。
   - 降低 GitHub Pages 或瀏覽器載入舊版資源的機率。
